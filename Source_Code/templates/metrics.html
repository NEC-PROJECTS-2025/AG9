Model Evaluation Metrics

This page evaluates the performance of our machine learning model in predicting obesity levels. We include various metrics, visualizations, and comparisons to demonstrate the model's strengths and limitations.

Key Metrics

Metric Description Value
Accuracy Percentage of correctly classified cases among all predictions. 91%
Precision Proportion of true positive predictions out of all positive predictions. 89%
Recall (Sensitivity) Proportion of true positives correctly identified by the model. 87%
Specificity Proportion of true negatives correctly identified by the model. 92%
F1 Score Harmonic mean of Precision and Recall. 88.5%
AUC (Area Under Curve) Measures the model's ability to distinguish between classes. 0.93
